Seccion 1. Participantes

Define quien envia, quien recibe y desde donde sale.

El sistema asigna el tracking automaticamente al crear el envio.
Si no eliges un almacen, el sistema intentara asignar uno automaticamente durante el despacho.

Seccion 2. Datos del envio

Peso, cantidad, volumen, prioridad y contexto comercial.

Si aplica, ingresa el dato en metros cubicos.

Seccion 3. Destino

Selecciona una direccion del destinatario o completa los datos manualmente.

Este desplegable muestra opciones guardadas del destinatario.
Usa una direccion del desplegable o completa direccion, ciudad, estado y codigo postal manualmente.
